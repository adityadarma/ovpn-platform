# Project Roadmap - VPN Platform

## Phase 1 --- MVP

Features:

-   basic manager API
-   simple agent
-   openvpn integration
-   create vpn user
-   generate .ovpn config

------------------------------------------------------------------------

## Phase 2 --- Management UI

Features:

-   admin dashboard
-   session monitoring
-   user management UI

------------------------------------------------------------------------

## Phase 3 --- Security

Features:

-   mTLS manager ↔ agent
-   RBAC permission
-   audit logging
-   rate limiting

------------------------------------------------------------------------

## Phase 4 --- Scalability

Features:

-   multi-node VPN
-   load balancing
-   node auto registration

------------------------------------------------------------------------

## Phase 5 --- Advanced Features

Features:

-   WireGuard support
-   multi-region nodes
-   network policy UI
-   monitoring dashboard
